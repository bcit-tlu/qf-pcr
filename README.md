# QF-PCR

Quantitative Fluorescence PCR (QF-PCR) online learning module.

## Development

```bash
npm install
npm start
```

## Docker

```bash
# Development (hot-reload)
docker compose up

# Production-like (nginx)
docker compose -f docker-compose.prod.yml up --build
```
